# 첫 번째 스테이지 (Mission 1)

## 개요

플레이어가 처음 마주하는 튜토리얼 성격의 스테이지. 게임의 핵심 규칙(이동,
작살 발사, 풍선 분열)을 자연스럽게 익힐 수 있도록 난이도를 낮게 구성한다.

## 구성 (초안)

- **풍선 구성**: 큰 풍선 1~2개로 시작 (분열을 거쳐도 화면이 과밀하지 않도록)
- **장애물**: 없음 또는 최소한의 단순한 플랫폼 1개 정도만 배치
- **제한 시간**: 넉넉하게 설정 (클리어 실패 페널티 체감 최소화)
- **아이템**: 기본 아이템 1개 정도 등장시켜 아이템 시스템을 소개

## 목표

- 플레이어가 좌우 이동과 작살 발사 조작을 익힌다.
- 풍선이 맞으면 분열한다는 규칙을 체험한다.
- 모든 풍선을 제거하면 스테이지가 클리어된다는 것을 학습한다.

## 클리어 조건

- 화면 내 모든 풍선(분열된 풍선 포함)을 제거하면 스테이지 클리어.

## 범위 메모

- 첫 스테이지는 장애물 반사 규칙 등 복잡한 요소는 최소화하고,
  이후 스테이지에서 장애물/다중 풍선 등 난이도를 점진적으로 높인다.
