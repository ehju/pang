# 게임 룰 (Game Rules)

PRD([../PRD.md](../PRD.md))에 정의된 핵심 컨셉을 바탕으로 한 게임 규칙 초안이다.

## 승리/패배 조건

- **스테이지 클리어**: 화면 내 모든 풍선을 제거하면 다음 스테이지로 진행한다.
- **목숨 감소**: 풍선에 캐릭터가 닿으면 목숨을 하나 잃는다.
- **게임 오버**: 목숨을 모두 소진하면 게임이 종료된다.

## 조작

- 좌/우 이동: 방향키 또는 A/D
- 작살 발사: 스페이스바 (또는 지정 키)

## 풍선 규칙

- 풍선은 중력의 영향을 받아 낙하하며, 바닥/벽/장애물에 부딪히면 반사되어 튕긴다.
- 풍선은 크기 단계(대/중/소)를 가지며, 작살에 맞으면 다음 단계로 분열한다.
  - 대 → 중 2개, 중 → 소 2개, 소 → 소멸
- 가장 작은 풍선을 맞추면 사라지고 점수를 획득한다.

## 장애물 규칙

- 장애물(벽, 플랫폼 등)에 닿은 풍선은 터지지 않고 반사각으로 튕겨나간다.
- 장애물은 작살로 파괴할 수 없으며, 작살은 장애물에 닿으면 그 지점에서 막힌다.

## 아이템 규칙

- 풍선 제거 시 확률적으로 아이템이 드롭된다.
- 아이템은 바닥까지 서서히 떨어지며, 낙하 중에는 사라지지 않는다.
- 아이템이 바닥에 닿으면 그 즉시 사라진다 (닿기 전에 습득해야 한다).
- 아이템 효과 예: 작살 강화, 이동 속도 증가, 일시 무적, 추가 점수/목숨 등.

## 시간 제한

- 스테이지별 제한 시간이 존재하며, 시간 내 클리어하지 못하면 페널티(풍선 추가 생성 등)가 발생할 수 있다.

## 범위 메모

- 초기 버전은 목숨 시스템 + 풍선 분열 + 장애물 반사까지만 구현하고,
  아이템/시간 제한 페널티는 이후 단계에서 확장 검토한다.
