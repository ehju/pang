# 메인 화면 (Main Screen)

## 개요

게임 실행 시 최초로 보여지는 화면으로, 플레이어가 게임을 시작하거나
설정을 확인할 수 있는 진입점 역할을 한다.

## 구성 요소 (초안)

- 게임 타이틀 로고
- 시작하기(게임 시작) 버튼
- 조작법 안내 (선택)
- 최고 점수 / 최근 기록 표시 (선택, 추후 확장)
- 설정(사운드 on/off 등) 진입 (선택, 추후 확장)

## 동작 흐름

1. 게임 로드 후 메인 화면이 표시된다.
2. 플레이어가 시작하기를 선택하면 첫 번째 스테이지(스테이지 1)로 진입한다.
3. (확장) 조작법/설정 화면은 메인 화면 위에 오버레이 또는 별도 화면으로 표시된다.

## 범위 메모

- 초기 버전은 타이틀 + 시작하기 버튼 정도의 최소 구성으로 시작한다.
- 최고 점수 저장, 설정 화면 등은 이후 스프린트에서 확장 검토.
